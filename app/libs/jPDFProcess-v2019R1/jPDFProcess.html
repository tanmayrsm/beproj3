Welcome to jPDFProcess


jPDFProcess is a Java library that can read, manipulate and then save or print PDF documents. With jPDFProcess, you can deliver customized PDF content to your users and customers by integrating within your servers and applications. jPDFProcess is platform independent, so it can be used in any environment that supports Java, including Windows, MacOSX and Linux.


This program demonstrates how easy it is to use jPDFProcess from your Java application. The program demonstrates some basic functions on PDF documents that can be done using jPDFProcess: merging, interleaving, printing, "stamping" and exporting a document to JPEG files. The code to do these functions can be found in the Sample class installed in a folder called jPDFProcessSamples under the folder where the jPDFProcess demo was installed.


Key Features of jPDFProcess are:


- Create new PDF documents or work with existing PDF documents

- Draw onto existing or new pages

- Split and merge documents

- Remove/ add / rotate / resize pages

- Encrypt and set document permissions

- Print PDF documents

- Export pages in a document as images

- Fill, import, export interactive forms fields data

- Apply and verify digital signatures

- Linearize PDF documents

- Add file attachments

- Add and edit bookmarks

- Edit document properties such as keywords, title, subject

- Create PDF layers and draw onto them

- Works on Windows, Unix and Mac OSX


Jar files - There are a number of jar files that are included in the installation of this evaluation program. Only the jPDFProcess.jar file is always required for deployment, the remaining jar files are used to work with specific features in some PDF documents:


jPDFProcess.jar - This is the main jar file for the component, it is always required.


bcmail.jar and bcprov.jar - These files contain the Bouncycastle security libraries, they are only used to verify digital signatures, when present.


cmaps.jar - This jar file contains CMaps, used to read and display character encodings used with CJK (Chinese, Japanese, Korean) content.


jai_imageio.jar - This jar file contains Java's ImageIO image decoders, including a decoder for JPEG 2000 images. This is only used when a document contains JPEG 2000 images.


jai_codec.jar - This jar file contains additional image encoders / decoders for ImageIO, it is necessary when opening and saving TIFF files.


rhino.jar - This jar file contains the Rhino JavaScript engine, it is necessary to run JavaScript for field calculation, validation and formatting.
